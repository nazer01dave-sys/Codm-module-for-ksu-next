# COD Garena Performance Module + Toast

- Aggressive CPU boost + Toast notification
- KernelSU/Next compatible
- Logs at `/storage/emulated/0/cod-performance.log`
- Flashable via KSU-next

## Installation
1. Run the build script or download ZIP from Releases
2. Flash via Magisk

   note: tested in Tecno Camon 30 5g(CL7) with mahiru v.5.1.1 kernel
